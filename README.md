# ZLStatusBarHUD
简单易用的信息显示状态栏
## 显示成功信息
''''objc

[ZLStatusBarHUD showSuccessWithMessage:@"加载成功"];

''''

## 显示失败信息
''''objc

[ZLStatusBarHUD showErrorWithMessage:@"加载失败"];

''''

## 显示普通信息
''''objc

[ZLStatusBarHUD showSuccessWithMessage:@"加载成功"];

''''

##  显示加载信息
''''objc

[ZLStatusBarHUD showMessage:@"自定义文字" image:nil];

''''
